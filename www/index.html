StackCheck

Your machine setup, verified in seconds.

npx stackcheck Zero-install, one-line setup checker
GitHub • npm

What StackCheck Validates:

  • • Node.js version
  • • Docker daemon status
  • • Required environment variables
  • • Port conflicts (e.g., 3000, 5432)
  • • Hardcoded secrets in .env
  • • Custom shell commands (CI prep, DB checks, etc.)

Sample Config

  {
    "node": ">=18.0.0",
    "docker": true,
    "env": ["DATABASE_URL", "JWT_SECRET"],
    "ports": [3000, 5432],
    "custom": ["./scripts/check-prereqs.sh"]
  }

How to Use

  1. 1 Add .stackcheck.json to your project
  2. 2 Run npx stackcheck anytime
  3. 3 Share config with your team to standardize setup
